Ejercicio 2 - Javascript

En este ejercicio se plantea un simple test, que deberá ser evaluado haciendo uso del lenguaje Javascript.

Cuando se pulse el botón Corregir se deberá hacer una serie de validaciones:

  • Cada pregunta debe tener una respuesta. En el caso que no seleccione una respuesta por pregunta, se mostrará un mensaje indicando "Debes seleccionar una respuesta para cada pregunta".
  • Las respuestas correctas son: Pregunta 1: Opción 2, Pregunta 2: Opción 1, Pregunta 3: Opción 3
  • Si todas las respuestas están contestadas, en el campo "Respuestas Correctas" se mostrará el número de respuestas acertadas.
  • Si todas las respuestas están contestadas, en el campo "Puntuación" se mostrará la puntuación obtenida en el test, teniendo en cuenta que se puntua cada respuesta acertada con 2 puntos.
